# F1 Dashboard 🏎️

I built this because I wanted to learn React properly, and what better 
way than tracking something I actually care about, like F1.

It pulls live data from the Jolpica F1 API (free, no key needed) and 
displays the current Driver and Constructor Championship standings.

## Tech Stack
- React + Vite
- React Router
- Jolpica F1 API

## Run it locally

git clone https://github.com/AlexKotkin/f1-dashboard.git
cd f1-dashboard
npm install
npm run dev
